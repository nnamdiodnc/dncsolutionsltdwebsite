# DNC Solutions Limited — Website

Static website for DNC Solutions Limited.

## Pages
- `index.html` — Home page
- `privacy-policy.html` — Privacy Policy
- `terms.html` — Terms of Service
- `health-and-safety.html` — Health & Safety Policy
- `coshh.html` — COSHH Assessment Policy

## Deployment
This site can be deployed directly via GitHub Pages:
1. Push this folder to a GitHub repository
2. Go to Settings > Pages
3. Set source to the root of your main branch
4. Your site will be live at `https://yourusername.github.io/repo-name`
